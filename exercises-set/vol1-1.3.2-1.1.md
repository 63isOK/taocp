# vol1 1.3.2 习题包1,习题1

## 题目

"X EQU 100" 并不是设置变量的指令.
如果要将某个值设置到具体的内存单元(X),
这个值是1000,MIXAL该怎么写.

## 分析

这个是要将1000写到内存单元X中

## 解法

- 先设置寄存器,后设置内存单元
  - VALUE EQU 1000, LD1 VALUE, ST1 X
  - LD1 =1000=, ST1 X
  - ENT1 1000, ST1 X
- 利用伪指令
  - ORIG X, CON 1000

上面3种写法,ENT1比LD指令有效率, 伪指令CON会更加方便.

书本上的demo大量使用CON来直接处理内存单元的写.
